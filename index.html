    技术文档

    Git Book

    Git是一个开源的分布式版本控制系统，可以有效、高速的处理从很小到非常大的项目版本管理。也是Linux Torvalds为了帮助管理Linux内核开发而开发的一个开放源码的版本控制软件

    Redis Documentation

    Redis(Remote Dictionary Server),是一个开源的使用ANSI C语言编写、支持网络、可基于内存亦可支持持久化的日志型、Key-Value数据库。

    npm中文文档

    NPM的全称是Node Package Manager，是一个NodeJS包管理和分发工具。

    Apache Dubbo

    Dubbo是阿里巴巴公司开源的高性能优秀、轻量级的Java RPC框架，它提供了三大核心能力：面向接口的远程方法调用，职能容错和负载均衡，以及服务注册和发现

    Apache Zookeeper

    Zookeeper是一个分布式，开源的分布式应用程序协调服务，是基于Google的Chubby的开源实现，也是Hadoop和Hbase的重要组件，它是一个为分布式应用提供一致性服务的中间件。

    JDK1.8 参考文档

    JDK是Java语言的软件开发工具包，它是Java开发的核心，包含了Java的运行环境和Java工具

    Hutool参考文档

    Hutool是一个小而全的Java工具类库，通过静态方法封装，降低相关API的学习成本以及提高工作效率。

    Nacos中文文档

    Nacos是阿里巴巴开源的用来实现配置管理及服务管理的中间件，能够帮助开发者更加敏捷容易的构建、交付、管理微服务平台。

    SEATA 中文文档

    Seata是一款开源的分布式事务解决方案，致力于提供高性能和简单易用的分布式事务服务。

    Kafka 官方文档

    Kafka是Apache基金会孵化的一个开源的流处理平台，它可以提供高吞吐量的分布式发布定于消息系统。

    WebPack中文文档

    webpack是一个现代JavaScript应用程序的静态模块打包器（module bundler）。

    Apache RocketMQ

    RocketMQ是一款开源的分布式消息中间件，由阿里巴巴开源，现已捐给了Apache基金会。

    分库分表ShardingSphere文档

    Apache ShardingSphere是一套开源的分布式数据库解决方案组成的生态圈，由JDBC、Proxy和Sidecar这3款组件组成。

    Netty中文文档

    Netty是由JBOSS提供的一个Java开源框架，提供了异步、事件驱动的网络应用程序框架和工具，用来快速开发高性能、高可用的网络服务器和客户端程序。

    Spring Cloud Netflix官方文档

    Spring Cloud Netflix是对Netflix开发的一套分布式服务框架的封装，包括服务的发现和注册，负载均衡、断路器等。
